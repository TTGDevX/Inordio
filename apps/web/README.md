# Web App

This is the main Next.js application. It will be initialized with:

```bash
cd apps/web
pnpm create next-app . --typescript --tailwind --eslint --app --src-dir
```

Then configured with:
- Shadcn/UI components
- tRPC for API
- Better-Auth for authentication
- Tailwind CSS for styling

See the root README for full setup instructions.
